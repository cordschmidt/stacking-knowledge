# Gitmoji Guide for Commit Messages

## ✨ Feature
**:sparkles:**

Implement a new model, training logic, utility function, or pipeline component

## 🐛 Bug Fix
**:bug:**

Fix a bug in the training process, data pipeline, evaluation logic, etc.

## 🧪 Experiment
**:test_tube:**

Run or log experiments (e.g. with new configs, models, W&B runs)

## 📝 Docs / Comments
**:pencil:**

Add or update documentation, README, or inline code comments

## ✅ Test / Eval
**:white_check_mark:** 

Add or update tests, metrics, or evaluation scripts

## ♻️ Refactor
**:recycle:**

Clean up or restructure code without changing functionality (e.g., modularizing training code)

## 📊 Data / Preprocessing
**:bar_chart:** 

Add or update datasets, data loading, preprocessing, or tokenization code

## 📦 Dependencies / Env
**:package:** 

Add or update dependencies, environment configs, or setup scripts

## ⚡ Performance / Hardware
**:zap:**

Improve performance (e.g., GPU utilization, data loading speed, memory usage)

## 🔒 Security / Privacy
**:lock:**

Address access controls, API keys, or privacy issues (like handling sensitive data)
